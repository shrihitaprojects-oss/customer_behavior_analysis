## ▶️ How to Run

### Prerequisites

* Python (>= 3.x)
* PostgreSQL
* Power BI Desktop

### Steps

1. Clone the repository:

   ```bash
   git clone <repo-link>
   cd <project-folder>
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the Jupyter Notebook for EDA:

   ```bash
   jupyter notebook
   ```

4. Set up PostgreSQL:

   * Create database
   * Import cleaned dataset
   * Run SQL scripts provided

5. Open Power BI file:

   * Load dataset
   * Refresh visuals
